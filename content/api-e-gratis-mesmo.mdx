author: 'Alisson Leal'
title: 'A API é realmente gratuita?'
---

Sim, a API é gratuita para uso atualmente e não possui limitações em termos de uso. No entanto, é importante observar que, no futuro, podem ser introduzidos planos de assinatura ou modelos de licenciamento para atender às necessidades de sustentabilidade e expansão do projeto. No momento, você pode aproveitar os recursos da API gratuitamente.
